# Gift Recommendation App 🎁

This is a practice project completely built with my prompting with ChatGPT 4o. This is a web application that provides personalized gift recommendations based on the recipient's name, interests, and budget. The app uses pre-built rules (not AI API) to generate dynamic and creative gift ideas.

---

## Features ✨

- **Gift Suggestions**: Uses pre-built rules to generate recommendations.
- **Customizable Inputs**: Allows users to specify the recipient's name, interests, and budget.
- **Responsive Design**: Built with Next.js and styled using TailwindCSS for a modern and clean user experience.

---

## Installation & Setup ⚙️

Follow these steps to run the app locally:

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS version recommended)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/gift-recommendation-app.git
   cd gift-recommendation-app
2. Install dependencies:
   ```bash
   npm install
4. Run the development server:
   ```bash
   npm run dev
6. Open your browser and go to:http://localhost:3000
